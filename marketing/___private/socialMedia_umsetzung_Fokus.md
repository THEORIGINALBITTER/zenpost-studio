# ◉ Social Media Kanäle Fokus Umsetzung -- 22.11.2025

| Kanal | Zielgruppe | Content-Strategie | Umsetzung / Frequency | Tools / Tipps |

| Twitter / X | Entwickler, Early Adopters, Tech-Community | - Screenshots, Mini-Tutorials, Launch-Updates- Storytelling: “Ich baue ZenPost alleine” | 3-5 Posts/Woche + 1 Launch Thread | Buffer / Hootsuite für Planung; Threads in 280 Zeichen klar |

| LinkedIn | Professionals, kleine Teams, Agencies | - Value Posts: Problem → Lösung → ZenPost- Beta / Early Access Updates | 2 Posts/Woche | Canva für Grafiken, PDF Screenshots, SlideShare |

| Reddit | Entwickler & SaaS Communities | - r/SideProject, r/opensource, r/ProductHunt- AMA / Feedback-Threads | 1 Post/Woche | Community-Regeln beachten; nicht zu werblich |

| Product Hunt | Early Tech Adopters | Launch + Updates | 1 Post Launch, 1-2 Updates | Hochwertige Screenshots + Video, Kommentar-Interaktion |

| YouTube / Shorts | Content Creators, Teams | - 1-min Tutorials, Quick Wins, Launch Story | 1–2 Shorts/Woche | Open-Source Logo, Subtitles, leicht verständlich |

| Discord / Slack | Community-Building | - Q&A, Beta Feedback, Feature Requests | Täglich moderieren | Kanäle klar trennen (Beta, Support, Chat) |

| Medium / Blog | SEO + Traffic | - “Journey: SaaS alleine bauen”- Tutorials, Best Practices | 1 Artikel/Woche | Blogposts cross-posten auf LinkedIn & Reddit |
