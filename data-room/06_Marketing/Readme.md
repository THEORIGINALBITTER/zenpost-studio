# ZenPost Studio

> Local-first AI-powered Publishing & Documentation Platform  
> Built for engineers, educators and teams who value clarity over noise.

---

## What ZenPost Studio Is

ZenPost Studio ist eine modulare Arbeitsplattform für:

- Schreiben
- Transformieren (AI)
- Dokumentieren
- Strukturieren
- Exportieren

Ohne Cloud-Zwang.  
Ohne Plattformabhängigkeit.  
Ohne Daten-Silos.

---

## Platform Architecture

ZenPost Studio folgt einem klaren Schichtenmodell:

### 1. UI Layer
- Markdown Editor
- Studio Navigation
- Export & Transformation Controls

### 2. Domain Layer
- Document Model (Markdown-first)
- Transform Engine (Rewrite, Summarize, Convert)
- Project Model (Doc Studio / Data Room)

### 3. Provider Layer
- AI Provider (Ollama local, erweiterbar)
- Storage Provider (Local Filesystem)
- Export Provider (Markdown/Text pipelines)

### 4. Integration Layer
- Tauri Desktop Shell
- Permission Model
- File I/O Control

---

## Studio Modules

### ZenPost Studio (Core)
- Local Markdown Editor
- AI-assisted Writing (local-first)
- Clean Export Pipelines
- Platform-independent output

### Doc Studio (Data Room Engine)
- Strukturierte Projektablage
- Technik- & Stack-Dokumentation
- Entscheidungs- & Notizsystem
- Langfristig lesbare Projekträume

### Planning Layer
- Aufgaben & Status
- Klar sichtbare Fortschritte
- Bewusst reduziert

## Design Philosophy

ZenPost Studio basiert auf drei Prinzipien:

- **Local-first**
- **Format-first**
- **Clarity-first**

Software sollte sich dem Denken anpassen.  
Nicht umgekehrt.
