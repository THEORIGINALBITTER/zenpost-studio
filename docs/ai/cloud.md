# Cloud-Provider

Cloud-Provider sind eine schnelle Alternative zur lokalen KI.

## Unterstuetzte Provider

- OpenAI
- Anthropic
- Custom API

## Einrichtung

1. API-Key erstellen
2. In den Einstellungen speichern
3. Modell auswaehlen

## Hinweise

- Kosten entstehen pro Anfrage
- Rate Limits beachten
- Daten gehen an den Provider
