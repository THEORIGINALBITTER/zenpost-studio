# KI – Übersicht

ZenPost Studio unterstuetzt lokale und Cloud-Provider. Beide Wege sind vollwertig.

## Lokal

- Volle Kontrolle
- Keine externen Requests
- Funktioniert offline

Typisch: Ollama

## Cloud

- Schnelle Einrichtung
- Gute Modellqualitaet
- Abrechnung pro Anfrage

Typisch: OpenAI, Anthropic

## Auswahl

- Datenschutz wichtig: lokal
- Zeit sparen: Cloud
- Mischbetrieb: moeglich
