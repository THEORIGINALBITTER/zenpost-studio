# Seiten exportieren

Mit Pages Export erzeugst du statische Seiten aus Markdown.

## Zweck

- Landingpages
- Produktseiten
- Dokumentationsseiten

## Ablauf

1. Markdown vorbereiten
2. Export starten
3. Zielordner waehlen

## Hinweise

- Saubere Ueberschriftenstruktur verwenden
- Bilder als relative Pfade einbinden
