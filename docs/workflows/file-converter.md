# Dateien konvertieren

Der File Converter bereinigt Inhalte und wandelt sie in saubere Formate.

## Inputs

- Markdown
- Editor.js JSON
- Rohtext

## Outputs

- Markdown
- HTML
- Text

## Ablauf

1. Datei waehlen oder Inhalt einfuegen
2. Konvertierung starten
3. Ergebnis pruefen und exportieren

## Hinweise

- Sonderzeichen werden bereinigt
- Ueberschriften bleiben erhalten
