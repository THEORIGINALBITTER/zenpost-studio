# Inhalte transformieren

Du wandelst einen Ausgangstext in ein plattformgerechtes Format um.

## Ablauf

1. Text einfuegen
2. Plattform waehlen
3. Ton und Laenge setzen
4. Transformieren

## Eingabe vorbereiten

- Kurze, klare Saetze
- Wichtige Fakten als Stichpunkte
- Technische Begriffe explizit

## Ergebnis pruefen

- Struktur stimmt
- Ton passt zur Plattform
- Fakten korrekt

## Wiederholen

Wenn das Ergebnis nicht passt, justiere Ton, Laenge oder Input und starte erneut.
