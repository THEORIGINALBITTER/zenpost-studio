# Allgemeine Probleme

## App startet nicht

Symptom

- App bleibt beim Start haengen
- Fenster bleibt leer

Loesung

- App neu starten
- System neu starten
- Falls wiederholt: Neuinstallation

## Export funktioniert nicht

Symptom

- Kein Output
- Zielordner bleibt leer

Loesung

- Zielordner pruefen
- Schreibrechte pruefen
- Export erneut ausloesen

## Dateien werden nicht erkannt

Symptom

- Datei erscheint nicht im Import

Loesung

- Format pruefen
- Datei zuerst in Markdown konvertieren
