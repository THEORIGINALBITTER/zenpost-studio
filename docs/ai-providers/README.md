# KI-Provider

ZenPost Studio unterstuetzt lokale und Cloud-Provider. Die Einstellungen findest du in der App unter KI.

## Lokal

- Ollama
- Laeuft auf deinem Rechner
- Kein externer Traffic

## Cloud

- OpenAI
- Anthropic
- Custom API

## Auswahl

- Datenschutz wichtig: lokal
- Schnelle Einrichtung: Cloud
- Mischbetrieb ist moeglich
