**The problem every content creator faces:** You write one great piece of content, but sharing it across LinkedIn, dev.to, Twitter, Medium, and Reddit means rewriting it 5+ times. Each platform has its own format, tone, and best practices.

## What is ZenPost Studio?

ZenPost Studio is a **free, open-source AI-powered content transformation tool** that helps developers, writers, and content creators adapt their content for any platform instantly.

**üéØ Key Features:**

- **Platform-Specific Transformations** - Convert markdown to LinkedIn posts, dev.to articles, Twitter threads, Medium stories, Reddit discussions, GitHub Discussions, or YouTube scripts
- **AI-Powered Intelligence** - Uses OpenAI, Anthropic Claude, or local Ollama models to understand context and adapt tone
- **Code Documentation Generator** - Turn code into professional README files automatically
- **Real-Time Markdown Editor** - Write and preview your content with live syntax highlighting
- **Beautiful Zen Interface** - Hand-drawn aesthetics with rough.js, dark theme, minimal design

**üíØ 100% Free & Privacy-First:**
- No backend - runs entirely in your browser
- Your API keys stay local (LocalStorage only)
- Open-source on GitHub
- No tracking, no data collection
- Free forever

## How It Works

### Step 1: Choose Your AI Provider

ZenPost Studio supports multiple AI providers:

**OpenAI (GPT-4, GPT-4o-mini)**
- Best for: General purpose, proven reliability
- Cost: ~$0.01-0.05 per transformation
- Setup: 2 minutes (get API key from platform.openai.com)

**Anthropic Claude (Claude 3.5 Sonnet)**
- Best for: High-quality, nuanced content
- Cost: ~$0.05-0.10 per transformation
- Setup: 2 minutes (get API key from console.anthropic.com)

**Ollama (Local AI - llama3.1, mistral)**
- Best for: Privacy, no costs, offline usage
- Cost: FREE (runs on your computer)
- Setup: 5 minutes (install Ollama, download model)

**Custom API**
- Best for: Enterprise, custom fine-tuned models
- Bring your own AI endpoint
